# Homebrew Tap for Victor AI

This tap contains the Homebrew formula for [Victor](https://github.com/vjsingh1984/victor), an enterprise-ready AI coding assistant.

## Installation

```bash
brew tap vjsingh1984/tap
brew install victor
```

## Upgrade

```bash
brew upgrade victor
```

## Uninstall

```bash
brew uninstall victor
brew untap vjsingh1984/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `victor` | Enterprise-Ready AI Coding Assistant |
